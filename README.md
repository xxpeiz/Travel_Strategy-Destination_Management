# Travel_Strategy-Destination_Management
(Java-based Full-stack Web Application)

Developed based on the Java, it mainly includes the design of WEB pages, database design, categorization of account information management permissions, and the implementation of various functional modules.
Front-end technologies used include JSP, CSS, and JS, built using the MVC development model. The database design uses MySQL, and the runtime environment adopts Apache Tomcat server.

基于 java 语言开发，主要包括 WEB 页面的设计，数据库设计和账户信息管理权限的分类以及各大功能模块的实现。
前端技术运用 JSP、CSS、JS，使用 MVC 开发模式进行搭建，数据库设计使用 MySQL 数据库，运行环境采用 Apache 的 Tomcat 服务器。
